# 💊 E-Pharmacy System – Frontend

## 📌 Overview
This is the **frontend application** for the E-Pharmacy System developed as part of the IETP5201 Integrated Engineering Team Project.  
The system helps users search for medicines, locate nearby pharmacies, and interact with pharmacy services through a modern and user-friendly interface.

---

## 🎯 Features
- 🔍 Search for medicines
- 📍 View nearby pharmacies (UI for location-based results)
- 🛒 Medicine details and availability display
- 👤 User authentication interface (Login/Register)
- 💳 Smart card (QR/RFID) integration interface (UI design)
- 📱 Responsive design for mobile and web
